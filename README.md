# 🔐 PHP Registration & Login System with Contact Form

This repository contains a simple **PHP-based authentication system** with registration and login functionality, plus a **contact form (send.php)** for sending messages.  
It includes front-end styling with CSS and interactivity/validation using JavaScript.

---

## 🚀 Features
- User registration with form validation
- Login system with session handling
- Secure password hashing
- Contact form (`send.php`) for sending messages
- Responsive design with CSS
- JavaScript validation for better user experience
